## How to use it
Unity-like controls.
WASDQE -> For moving arround the scene.
Right Click + Mouse movement -> FPS camera
Alt + Left Click -> Orbit arround the model.
MouseWheel -> Zoom
Dragging and Dropping an fbx to the scene will load the model with their textures.


Enjoy! :)
